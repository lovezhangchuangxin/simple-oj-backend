**技术栈**

- SpringBoot + MyBatis-Plus + MySQL
- [docker-java](https://github.com/docker-java/docker-java)

## 快速开始

1. 查看 `pom.xml` 文件，安装相关依赖。
2. 复制 `application-template.yml` 为 `application-dev.yml`，并修改其中的配置。
3. 运行 `SimpleOjBackendApplication` 类。

## 基本功能

目前几乎就登录注册，啥也没有。
